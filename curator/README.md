# Dugong framework

## Description

The Dugong framework.

## How-to

* Install the required libraries, run `npm install`
* Run the front-end development server, `npm run dev`
* Open the web app on `http://localhost:8000/`

## sudah_kudugong UI library
To connect `sudah_kudugong` UI library, run `npm link sudah_kudugong`
